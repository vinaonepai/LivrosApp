## 📖 Sobre o projeto

O **LivrosApp** é um aplicativo desenvolvido utilizando Ionic + Vue, com o objetivo de simular uma biblioteca digital interativa.

O sistema permite que o usuário navegue por uma lista de livros, visualize detalhes, realize login e cadastro, recupere senha e favorite seus livros preferidos.

O projeto possui um design moderno com estilo futurista, utilizando cores neon e efeitos visuais para melhorar a experiência do usuário.

---

## 🚀 Funcionalidades

* 🔐 Sistema de Login
* 📝 Cadastro de usuário
* 🔑 Recuperação de senha
* 📩 Tela de confirmação de recuperação
* 🏠 Página inicial com lista de livros
* 📖 Visualização de detalhes dos livros
* ❤️ Sistema de favoritos com persistência (localStorage)
* 🔄 Navegação entre páginas

## ⚙️ Como rodar o projeto

### 1. Instalar as dependências

```bash
npm install
```

### 2. Rodar o projeto

```bash
ionic serve
```

### 3. Acessar no navegador

O projeto será aberto automaticamente em:

```
http://localhost:8100
```

## 💾 Armazenamento

Os dados de favoritos são armazenados utilizando **localStorage**, permitindo que o usuário mantenha seus livros favoritos mesmo após recarregar a aplicação.

---

## 🎯 Objetivo acadêmico

Este projeto foi desenvolvido com o objetivo de praticar:

* Desenvolvimento com Ionic + Vue
* Navegação entre páginas (router)

* Persistência de dados no navegador
